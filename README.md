# Gestion-Des-Reservation-Swing

## Description
Ce projet est une application de gestion des réservations développée en Java en utilisant Swing pour l'interface utilisateur. Elle permet aux utilisateurs de créer, modifier, et supprimer des réservations, ainsi que de consulter les réservations existantes.

## Fonctionnalités
- Ajouter une nouvelle réservation
- Modifier une réservation existante
- Supprimer une réservation
- Consulter les réservations
- Rechercher des réservations par critères (date, nom, etc.)

## Configuration requise
- Java JDK 8 ou supérieur
- IDE recommandé : IntelliJ IDEA ou Eclipse

## Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Sohaib-Laarichi/Gestion-Des-Reservation-Swing.git
   ```
2. Importez le projet dans votre IDE.
3. Assurez-vous que votre JDK est configuré correctement.
4. Exécutez l'application depuis l'IDE.

## Utilisation
1. Lancez l'application.
2. Utilisez les menus et les boutons pour gérer les réservations.
